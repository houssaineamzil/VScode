## How to run locally?

Make sure you have [Node](https://nodejs.org/en/) and [git](https://git-scm.com/) installed.

- Clone the repo:

  ```bash
  git clone https://github.com/muhammedia/VScode.git
  cd VSCode
  ```

- Install the dependencies:

  ```bash
  npm install
  ```

- Fire up a development server:

  ```bash
  npm start
  ```
